== Installation ==

### Installation

1. Upload 'alobaidi-timeline' folder to the '/wp-content/plugins/' directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.


### Usage

Use this shortcode in post or page:
    [alobaidi_timeline]

### Shortcode Attributes

Alobaidi timeline has 4 attributes:

    number="here enter number of posts per page" default is "100".
    list="here enter list type, for example: ol" default is "ul".
    prev="here enter text translate" default is "Previous Page".
    next="here enter text translate" default is "Next Page".

For example (Spanish translation):

    [alobaidi_timeline number=”50″ list=”ol” prev=”Pagina anterior” next=”Pagina siguiente”]
